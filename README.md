Section 3: Short Answer

8. Describe the steps involved in conducting a network vulnerability assessment.



Answer:

Identify network assets (devices, servers, applications)

Scan the network using vulnerability scanning tools

Identify and analyze security weaknesses

Assess the risk level of vulnerabilities

Recommend and apply security fixes

Re-test the network to ensure vulnerabilities are resolved



---

Section 4: Practical Application

9. Demonstrate how to troubleshoot network connectivity issues using the ping command.



Answer:

Open Command Prompt / Terminal

Use the command: ping <IP address or website>

Check the response:

Successful replies indicate connectivity

Request timed out indicates network issues


Use results to identify packet loss or latency problems

Section 5: Long Answer

10. Discuss the importance of regular network maintenance and the key tasks involved in maintaining network infrastructure.



Answer:
Regular network maintenance is essential to ensure security, reliability, and optimal performance of network systems. It helps prevent downtime, security breaches, and performance degradation.

Key maintenance tasks include:

Regular software and firmware updates

Monitoring network performance and traffic

Managing backups and disaster recovery plans

Checking hardware health (routers, switches, cables)

Implementing security measures such as firewalls and antivirus updates

Documenting network configurations


Proper maintenance improves system efficiency, extends hardware lifespan, and ensures business continuity.
